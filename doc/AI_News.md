# AI要件定義書

## 1. 目的

SNS投稿を入力として、その投稿を題材としたニュース記事を自動生成する。

## 2. AI機能要件

### F-01: 投稿内容の解析

- 入力: 投稿本文、投稿者名、投稿日時、添付メディア有無
- 出力: トピック分類、要旨、感情極性、固有表現（人物/組織/地名）

### F-02: 記事生成

- 入力: F-01の解析結果 + 元投稿
- 出力: 見出し（30字以内）、投稿元(用意されているニュースアカウントの中でランダム)リード文（100字以内）、本文（400〜800字）、カテゴリタグ、サムネイル用キーワード
- 文体: 選択されたニュースアカウントのスタイル（オネエ、ゲイ、盲信者、ツンデレetc...）、投稿者を肯定し、誇張した表現の文

## 3. 非機能要件

